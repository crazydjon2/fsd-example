## Команды (Nuxt 4)

```bash
# Установка зависимостей
yarn
# или npm / yarn
npm install

# Разработка (HMR + devtools)
yarn dev
# → http://localhost:3000

# Сборка для продакшена
yarn build
# → создаёт .output/

# Локальный предпросмотр продакшена
yarn preview

# Линтинг и проверка типов
yarn lint
